#### Simple and easy tools for storing values while efficiently keeping them encrypted in the memory.
- Currnetly supports in C#  
  - **EncInt** (int)
  - **EncLong** (long)
  - **EncFloat** (float)
  - **EncDouble** (double)
  - **EncDecimal** (decimal)
  - **EncString** (string)
